# N빵의 민족

배달료 급증을 인한 배달료 줄이기 프로젝트

___

프로젝트 기간
- 2022/4/1 ~ 2022/6/30

### 🙋‍♂️ 역할

- **로그인 관리**: 안드로이드 앱에서 로그인하고 관리
- **구글 맵**: 구글 맵으로 배달 받고 싶은 위치 선택
- **메뉴선택 관리**: 선택한 메뉴와 시간 장소 관리

### 📖 설명

- **프로젝트 개요**: 배달료가 급등하고 혼자사는 사람들의 배달료 부담을 줄이기 위해서 주변에 같은 가게에 주문하고 싶은 사람들을 모아 배달료를 1/N하는 프로젝트
